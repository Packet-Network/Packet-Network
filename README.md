# Packet-Network (NetDesign)

ネットワーク設計を学べる教育ゲーム

## コンセプト

- **繋がっていれば即死しない** - ゲームオーバーではなくスコアで評価
- **設計が悪いほどスコアが伸びない** - トレードオフを学ぶ
- **用語・概念は現実準拠** - そのままプロの思考へ

## 機能

### Stage 1-2 (MVP)
- デバイス配置: PC、スイッチ、ルーター
- ケーブル接続（ポート制限あり）
- リアルタイム疎通チェック
- スコア評価: 接続性 / 効率性 / 拡張性 / 冗長性

### 予定
- ステージシステム
- VLAN設定 (Stage 3+)
- BGP / ECMP (Stage 5+)
- ランキング

## ローカル実行

```bash
python3 -m http.server 8000
# http://localhost:8000 でアクセス
```

## 設計思想

| レベル | テーマ | 学ぶこと |
|--------|--------|----------|
| 1 | つなぐ | ケーブル、ルーター、スイッチ |
| 2 | 分ける | VLAN、セグメント |
| 3 | 守る | 冗長化、障害ドメイン |
| 4 | 伸ばす | スケール、将来設計 |
